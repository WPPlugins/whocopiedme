== Description ==

[WhoCopiedMe](https://whocopied.me/ "The Who Copied Me Homepage") is a tool helping you to find out if your content has been copied and viewed on other webpages.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Signup at https://whocopied.me with the url of your blog
1. Go to the Who Copied Me dashboard and find out if your content get's stolen
